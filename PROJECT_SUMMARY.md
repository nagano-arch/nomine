## 🎨 デザイン思想

### 消費者向け
- **コンセプト**: 選ばれる体験
- **デザイン**: Instagramライク、今っぽい、高級感とカジュアルの中間
- **カラー**: ダークベース（#1a1a1a）+ アンバーアクセント（#f59e0b）
- **UX**: ログイン不要、3タップでエントリー完了

### 管理画面
- **コンセプト**: 飲食店でも迷わず使える
- **デザイン**: モダン、余白広め、大きめカードUI
- **カラー**: ホワイトベース + アンバーアクセント
- **UX**: 1画面で状況把握、直感的な選出フロー

## 💡 独自の技術的工夫

### 1. 営業日管理
24時をまたぐ営業（17:00-25:00など）に完全対応
- 前日23:59から翌日01:00を同一営業日として扱う
- QRコードは固定、営業日単位でアルバムが切り替わる

### 2. AI採点システム
- 公式素材としての品質を4軸で評価
- 業態別の補正係数
- 納得感のあるコメント生成ロジック

### 3. マルチテナントアーキテクチャ
- マスター管理者が複数テナントを管理
- テナント管理者が自店舗のみ管理
- 完全なデータ分離とアクセス制御

## 📦 デプロイ準備

### ローカル開発
```bash

npm run db:migrate:local
npx wrangler d1 execute nomine-production --local --file=seed.sql
```
